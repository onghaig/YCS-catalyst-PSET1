14.7 - Maximum and Minimum Values:

This section explores how to find and classify the maximum and minimum values of multivariable functions, also known as extremum points. In multivariable calculus, we are often interested in identifying where a function reaches its highest or lowest value in a given region. To do this, we use partial derivatives to find critical points, where the gradient (the vector of partial derivatives) is equal to zero or undefined. These points are candidates for local maxima, local minima, or saddle points.

  • Just as in single-variable calculus, we can find maximum and minimum values in 2d functions.
  • Absolute Max/Min- greatest values found in the whole function, requires that the function is closed and bounded (see later)
  • Local Max/Min - Peaks of min and maximum
  • Saddle points- points where depending on which direction you are going, are a maximum or a minimum
  • For a local maximum or minimum to occur, the partial derivatives must = 0. These are also called critical points

Second Derivative Test:

  • Given that second partial derivatives exist, (a,b) is a critical point (the partial derivatives in both x and y = 0),
  • Define the discriminant $D$ at $(a, b)$ as: \[ D = f_{xx}(a, b)f_{yy}(a, b) - (f_{xy}(a, b))^2 \]

The second partial derivatives test states:

  • If $D > 0$ and $f_{xx}(a, b) > 0$, then $f$ has a local minimum at $(a, b)$.
  • If $D > 0$ and $f_{xx}(a, b) < 0$, then $f$ has a local maximum at $(a, b)$.
  • If $D < 0$, then $f$ has a saddle point at $(a, b)$.
  • If $D = 0$, the test is inconclusive.

To test for local minimums or maximums: we can find critical points by setting the partial derivatives equal to zero and evaluating the second partial derivative test at these points.

Absolute Maximum and Minimum Values

  • In order to find absolute extrema, the function must be both a closed and bounded set
  • Closed: a closed set in R3 is one that contains all its boundary points
  • Bounded set - one that is contained within a possible disk, it is finite in extent.

How to find absolute extrema:

  1. Find the values of $F$ at the critical points of $f$ in $D$
  2. Find the extreme values of $F$ on the boundary of $D$
  3. Find the largest of those values from the first two steps; and smallest, those are min/max
